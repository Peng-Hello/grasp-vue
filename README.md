# grasp-vue

![MIT License](https://img.shields.io/github/license/Peng-Hello/grasp-vue)

[English version](./docs/en.md)

A toy for learning vue ！
如你所见的仓库名称一样,该仓库是学习 Vue 时的玩具。
## 快速开始
```shell
# 克隆本仓库
git clone git@github.com:Peng-Hello/grasp-vue.git
# 切换到对应仓库目录下
cd grasp-vue
# 安装依赖
pnpm install
# 运行单元测试
pnpm run test
```
## 技术栈
- TypeScript
- vitest (一个强大的单元测试框架)
